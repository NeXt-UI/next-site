  • Set direction for a single-link path

    Set direction for a single-link path

    Each link has a source and a destination. Sometimes this notation does not correspond to a path, that is the path's source could be the link's destination. If your path consists of a single link, you should specify the source node ("sourceNode" attribute) in path's configuration in order to display it in a correct way. In this example our path corresponds to link #2 (Houston -> New York), but we override it to make path be directed to Houston from New York.

    See the Pen Set direction for a single-link path by Aikepaer Abuduweili (@NEXTSUPPORT) on CodePen.
