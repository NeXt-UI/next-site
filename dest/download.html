Bower/npm installation

To install the framework with Bower, navigate to your project's folder and run in terminal:
bower install NeXt

For npm the syntax is slightly different:
npm install next-bower

0.8 Release note

Major Features

Beryllium release of NeXt project provides a network-centric topology UI that offers visualizations of the following:

  • Large complex network topologies supported
  • Aggregated network nodes functionality added
  • Traffic/path/tunnel/group visualizations added
  • Different layout algorithms added
  • Map overlays support added
  • Preset user-friendly interactions added

Target Environment

For Execution
  • Requires latest version of Chrome/Firefox
For Development
  • GNU
    • Coreutils
    • Findutils
    • SED
    • Make
  • NodeJS & NPM
    • PhantomJS
    • LESS
    • YUIDoc
    • Uglify

Known Issues and Limitations

  • No IE support in this version

0.5 Release note

Major Features

Topology
  • Added node-set support for aggregation (nested topology)
  • Added several new color themes (blue/green/black/navy)
  • Added customized node icon support
  • Added popup info panel for multiple links
  • Added basic touch events support
  • Improved algorithm of labels’ position to avoid occlusion
  • Added hint for link besides node hint

Bug Fixes

  • [Core] Fixed memory leak issue in event engine.
  • [Topology] Fixed dragging issue in IE9.
  • [Topology] Fixed problematic behaviors in selecting nodes.
  • [Topology] Fixed memory leak issue when resizing.

Known Issues and Limitations

  • Popup’s position may not be correct when displays in corners
  • Accessibility is not completed
  • Thumbnail cannot reflect the correct position of topology
  • Hints may overlap with nodes
  • Some style issues under IE 9.0
  • Zooming issue when node set is expanded

0.4 Release note

Major Features

Core
  • Added Vector/Line for easy calculation
  • SetAll/getAll are deprecated and are replaced with gets/sets for easy usage
  • NotifyPropertyChanged/listen are deprecated and are replaced with watch/unwatch/notify due to performance enhancement, easy to use and fixing issue of memory leak
Graphic
  • Graphic module is now using the same view engine and event engine as the UI module

Known Issues and Limitations

  • Popup’s position may not be correct when displays in corners
  • Accessibility is not completed
  • Cannot do multi-selection by keyboard
  • Problem when switching between topology’s scenes
  • Cannot use “Full-screen mode” in iframe
  • Dragging is not working in IE9
