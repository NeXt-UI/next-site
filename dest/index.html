Build with NEXT

NEXT is a front-end framework that maximizes the power of JavaScript and JSON and provides a powerful topology component to visualize networks.

Download NEXT

NEXT is an effective framework for network visualization

Use NEXT to build all types of visualization components to bring your network to life


NEXT FRAMEWORK comprises UI and CORE

NEXT TOPOLOGY component is a functional tool to enhance network experience. See Examples

NEXT UI platform supports reuseable UI components paradigm

NEXT CORE is the foundation of the framework, a supplement of JavaScript to leverage the advantage of MVVM

Efficient Framework

NEXT is designed and structured base on the concept of data driven. The adoption of MVC and MVVM improves productivity and maintainability. Plus, that's free and open-source.

Effective Topology

NEXT comes with Cisco's 30-year networking experience, and is user oriented. It is easy to build and visualize network topologies with NEXT just by following our free tutorials and checking out examples.

High Performance

NEXT framework is constructed with concise codes and accurate structures to achieve high performance.

Active Support

NEXT is under active development and the regular iterations and updates make the framework more stable and more reliable. We welcome newcomers and happy to collaborate with other organizations, companies and universities.
Contact the team

Rich APIs

NEXT provides a wide range of well-structured JavaScript APIs to enable users to efficiently and quickly achieve their technical goals, engineer customized topology and build networking applications on the web. View API.

Friendly & Compatible

NEXT is compatible with popular third party CSS and JS frameworks, such as Bootstrap, Angular and React, which makes creating beautiful and smart web apps really effortless.
