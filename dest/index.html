NEXT FRAMEWORK is comprised of UI and CORE

NEXT TOPOLOGY component is a functional tool to enhance network experience. View Examples

NEXT UI is a platform to support the making of reuseable UI components

NEXT CORE is the foundation of the framework, a supplement of JavaScript to leverage the advantage of MVVM

Efficient Framework

NEXT is designed and structured base on the concept of data driven. The adoption of MVC and MVVM improves productivity and maintainability.

Effective Topology

It is easy to generate sample topology with NEXT, and by investing minimum effort to follow the tutorials, users could leverage more advanced functionality like aggregation. View Example

High Performance

NEXT framework is constructed with concise codes and accurate structures to achieve high performance.

Active Support

NEXT is under active development and the regular iterations and updates make the framework more stable and more reliable. Contact the team

Rich APIs

NEXT provides a wide range of well-structured APIs to enable users to efficiently and quickly achieve functionalities, customised topology and build applications on the web. View API
